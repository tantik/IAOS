# IAOS Specification v0.1

Status: Draft
Owner: Konstantin / Izumi IT
Purpose: Define what IAOS is and what it is not.

## 1. Definition

IAOS is the company operating system for Izumi IT.

It defines how Izumi IT:
- makes decisions;
- builds products;
- uses AI;
- manages knowledge;
- sells and supports products;
- improves over time.

## 2. What IAOS is

- Company operating system
- Knowledge and process system
- AI-readable working standard
- Foundation for scaling Izumi IT

## 3. What IAOS is not

- Not a theoretical knowledge base
- Not a replacement for product repositories
- Not tied to one AI vendor
- Not a place to store generic internet knowledge

## 4. Current Mission

Help one founder build and operate high-quality SaaS products with AI support.

## 5. Target Mission

Help Izumi IT scale into a mature AI-native product company.

## 6. Core Principle

Build the ideal system progressively, starting with practical workflows that are useful today.
